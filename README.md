# Breef Bot

Телеграм-бот для проведения опросов (брифов) и управления клиентской базой с интеграцией Google Sheets.

## 🚀 Основной функционал

- **Стандартные команды:** `/start`, `/help`, `/menu` для базового взаимодействия.
- **Проведение опросов:** Автоматизированный сбор информации от пользователей через FSM (машину состояний).
- **Интеграция с Google Sheets:** Хранение и получение данных о клиентах и их ответах.
- **Административные функции:**
    - Отправка сообщений конкретным пользователям.
    - Массовые рассылки.
    - Просмотр посетителей за день.
- **Логирование:** Запись всех ключевых событий и ошибок в файл.

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Ihlop/breef-bot.git
cd breef-bot
```

### 2. Настройка окружения

1.  **Установите Python:** Рекомендуемая версия **3.12.3**. Убедитесь, что у вас установлен Python версии 3.11 или выше.
2.  **Установите `uv`:** `uv` используется для управления зависимостями. Инструкции по установке можно найти [здесь](https://github.com/astral-sh/uv).
3.  **Установите зависимости:**

    ```bash
    uv sync
    ```

### 3. Настройка конфигурационных файлов

1.  **Создайте файл `passwords.py`:** Этот файл хранит ваши секретные ключи.

    ```python
    # passwords.py
    codemachinee_breef_bot = "YOUR_TELEGRAM_BOT_TOKEN"
    loggs_acc = "YOUR_TELEGRAM_CHAT_ID_FOR_LOGS"
    codemashine_test = "YOUR_TEST_BOT_TOKEN" # опционально
    ```

2.  **Добавьте файл ключа Google Service Account:**
    - Получите JSON-файл с ключом сервисного аккаунта Google Cloud с доступом к Google Sheets API.
    - Переименуйте его в `pidor-of-the-day-af3dd140b860.json` (или измените имя в `google_sheets.py`) и поместите в корневую директорию проекта.

### 4. Запуск бота

```bash
uv run python main.py
```

## 🐳 Установка через Docker

### 1. Подготовка файлов

Перед сборкой образа убедитесь, что в корневой директории проекта находятся:
- Файл `passwords.py` (созданный по инструкции выше).
- Файл ключа Google Service Account (`pidor-of-the-day-af3dd140b860.json`).

### 2. Сборка Docker-образа

```bash
docker build -t breef-bot .
```

### 3. Запуск Docker-контейнера

```bash
docker run -d --name breef-bot-container breef-bot
```

## 📂 Структура проекта

```
├── Dockerfile              # Конфигурация для сборки Docker-образа
├── FSM.py                  # Определения состояний для FSM (Finite State Machine)
├── functions.py            # Функции для управления клиентской базой
├── google_sheets.py        # Логика для взаимодействия с Google Sheets
├── handlers.py             # Обработчики команд и сообщений от пользователей
├── keyboards.py            # Функции для создания Telegram-клавиатур
├── main.py                 # Главный файл, точка входа в приложение
├── pyproject.toml          # Файл с метаданными проекта и зависимостями
└── tests/                  # Тесты для проекта
```
